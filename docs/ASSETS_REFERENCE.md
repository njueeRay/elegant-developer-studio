## 素材规则

- 生产 UI 使用的素材放在 `public/assets`。
- 已采纳视觉参考放在 `public/references`。
- Codex 临时目录中的生成图不能被应用代码直接引用。
- 信息密度低、无法指导设计的截图不进入正式参考集。
- 后续新增媒体素材前，必须先明确它服务哪个页面、组件或叙事目标。
